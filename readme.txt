Простое приложение для заметок, построенное на Flask.

## Установка

### Предварительные требования
- Python 3.x
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- Flask-Login

### Инициализация базы данных
bash
flask db init
flask db migrate
flask db upgrade

### Запуск приложения
bash
python app.py

Доступ к приложению по адресу `http://127.0.0.1:5000`.

## Функции
- Регистрация пользователей
- Вход в систему
- Создание заметок
- Редактирование заметок
- Удаление заметок

## Используемые технологии
- Flask
- SQLAlchemy
- SQLite
- Flask-Login
- Jinja2
